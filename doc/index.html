我们的故事

传承礼尚往来，铭记温暖情谊

礼尚往来，是中华民族的传统美德，老一辈人一般会在举行宴席的时候用手写人情簿的方式来记录每一个亲友的送礼。这种方式存在很多弊端，比如人情簿丢失、无法携带在身边、很难搜索到某个人的送礼记录、由于手写原因无法辨认清楚内容等等。

于是，一群热爱生活的开发者与设计师携手同行，打造了这个将传统人情记账与现代科技深度融合的开源项目。我们坚信，让古老的人情往来智慧与现代数字技术碰撞交融，方能让传统礼尚往来文化在数字时代重绽华彩。

前端使用指南

快速开始前端开发

按照以下步骤快速搭建和运行礼记前端项目

环境要求

  • Node.js 16.0 或更高版本
  • pnpm 包管理器（推荐使用）
  • Git 版本控制工具
1

克隆项目仓库

从 GitHub 克隆项目到本地

git clone https://github.com/Chadwuo/li-ji-weapp.git
cd li-ji-weapp
2

安装依赖

使用 pnpm 安装项目依赖，速度更快且节省磁盘空间

pnpm install
3

启动开发服务器

在本地运行开发环境，进行前端开发调试

pnpm dev
4

构建生产版本

打包生产环境代码，用于部署到服务器

pnpm build
