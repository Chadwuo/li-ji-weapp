### 上手指南

###### **克隆项目**
```bash
git clone https://github.com/micahh28/hhgiftcash.git
```

###### **前端WebApp项目**

```bash
# 安装依赖
npm install

# 本地服务 localhost:8080
npm run serve

# 构建生成环境
npm run build
```

###### **后端WebApi项目**

```bash
#  Visual Studio 打开项目
WebApp项目由Visual Studio 2019开发

# Visual Studio运行本地服务 
localhost:****

# 发布项目至ISS
```
